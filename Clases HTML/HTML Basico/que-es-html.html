HTML = HyperText Markup Language. Es el lenguaje de marcado que se utiliza para crear páginas web. HTML describe la estructura de una página web utilizando etiquetas y atributos. Las etiquetas HTML son elementos que indican al navegador cómo debe mostrar el contenido. Por ejemplo, la etiqueta

se utiliza para definir un párrafo, mientras que la etiqueta

se utiliza para definir un encabezado de nivel 1.

HTML es un lenguaje de marcado, lo que significa que se utiliza para estructurar el contenido de una página web. HTML no es un lenguaje de programación, ya que no tiene lógica ni funciones. En su lugar, HTML se utiliza junto con otros lenguajes como CSS (Cascading Style Sheets) y JavaScript para crear páginas web interactivas y atractivas.
